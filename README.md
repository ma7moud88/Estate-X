# EstateX 🏡

**EstateX** is a modern digital platform specialized in buying and selling lands. It aims to connect buyers and sellers in a simple, fast, and secure environment.

### 📍 Local Focus
The platform's first phase focuses on **New Qena (قنا الجديدة)**, providing users with precise details regarding location, area, land type, and pricing.

---

## 🚀 Key Features
* **Simplified UI:** Easy exploration of available lands.
* **Seller Tools:** Easy listings with image uploads and precise location tagging.
* **Advanced Search:** Smart filters to find the best opportunities.
* **Trust & Security:** Account verification and transparent data display.

## 🛠️ Tech Stack
* **Framework:** Next.js (App Router)
* **Language:** TypeScript
* **Styling:** Tailwind CSS
